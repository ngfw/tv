# Fun With Roku & Plex

### Installation
```bash
git clone https://github.com/ngfw/tv.git
```
then edit `settings.json`.

That's it.


### Roku Volume Down:
```bash
./TV.volume.down
```
or
```bash
./TV.volume.down 20
```
### Roku Volume Up:
```bash
./TV.volume.up
```
or
```bash
./TV.volume.up 20
```

### Plex Play Random top movie:
```bash
./TV.plex.topmovie
```
